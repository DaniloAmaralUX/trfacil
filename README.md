# TR Fácil

Aplicação standalone para criação, revisão e acompanhamento de Termos de Referência do Sistema FIEPE.

## Fluxos incluídos

- Dashboard do TR Fácil
- Listagem de TRs
- Wizard de criação de novo TR
- Visualização de TR
- Fila de aprovações

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Observações

- Este repositório foi extraído do app original para manter apenas o produto TR Fácil.
- A autenticação ficou fora desta rodada; o app roda em modo local/mockado.
- O wizard mantém os modelos refinados para FIEPE/IEL e SESI.
